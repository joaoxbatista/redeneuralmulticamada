[Problema:]
Verificar se existem duas entradas ligadas no conjunto de 4 entradas.
Caso existam duas entradas ligadas retorna 1, caso contrário retornar 0.

----------------------------------------------------------------------------------------

[Estrutura das pastas do projeto:]

app - Onde ficam os arquivos da aplicação
	-> classes: Contém as classes da rede neural
		-> MultilayerNetwork.py: Classe "abstrata" que define os métodos de um rede multicamada
		-> SigmoidMultLayerNetwork.py: Classe para representar uma rede multicamada sigmoid 

	-> Main.py: Tela principal
	-> Trainer.py: Tela de treinamento

datasets - Conjunto de treinamento
	-> train-input.csv: Conjunto de treinamento apenas com as entradas
	-> train-output.csv: Conjunto de treinamento apenas com as saidas
